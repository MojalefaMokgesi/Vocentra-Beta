# Vocentra Beta

Vocentra Beta is a web-based job application and recruitment platform built with ASP.NET Core MVC.  
It is designed to simplify how candidates apply for jobs and how employers manage applications.

This repository represents the **beta version** of the Vocentra platform and is under active development.

---

##  Features

- Job listings with detailed views
- Job application workflow
- Applicant tracking
- Clean MVC architecture
- Entity Framework Core integration
- Responsive UI for desktop and mobile

---

##  Tech Stack

- **Backend:** ASP.NET Core MVC
- **Frontend:** Razor Views, HTML, CSS, JavaScript
- **Database:** Entity Framework Core (SQLite / SQL Server ready)
- **Authentication:** ASP.NET Identity (planned / partial)
- **Version Control:** Git & GitHub
